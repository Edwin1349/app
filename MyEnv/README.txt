# Install python version 3.5.0 or higher:
# Install pip:
	1.Download get-pip.py to a folder on your computer.
	2.Open a command prompt and navigate to the folder containing get-pip.py.
	3.Run the following command

	   python get-pip.py

	You can verify that Pip was installed correctly by opening a command prompt and 	entering the following command:

	pip -V
# Now is the installation done without errors you can install the project
# go to the directory where you have put the project
         pip install -r requirements.txt
# go to the main directory of your project and run the following command
        python manage.py runserver
#open http://localhost:8000/ in your browser
